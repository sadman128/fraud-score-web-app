## 📌 About

**FraudScore** is a full-stack social media platform for **fraud awareness** and community discussion. Users create posts, share fraud experiences, view detailed fraud profiles with risk scores, and engage through likes/dislikes/comments.

Production-ready **Spring Boot 3.x** (Java 21) backend + **React 18** (Vite + Tailwind CSS) frontend with JWT authentication and MongoDB Atlas.

> 💡 Perfect for fraud detection teams, cybersecurity communities, awareness campaigns.

---

## ✨ Features

### 👥 User System
- JWT Authentication + Refresh Tokens
- Profile Management
- Role-based Access (USER/ADMIN)

### 📱 Social Features
- Posts CRUD (Create/Edit/Delete)
- Like/Dislike Reactions
- Post Comments System
- Search & Trending Pages
- Infinite Scroll Feed

### 🔍 Fraud Intelligence
- Detailed Fraud Profile Pages
- Risk Scoring Visualization
- Case Study Analysis
- Admin Content Moderation

### 👨‍💼 Admin Dashboard
- User Management
- Post Moderation
- Analytics Overview

### 🎨 UI/UX
- Responsive Tailwind Design
- Dark/Light Mode Toggle
- Toast Notifications
- Lucide React Icons
